GIF 파일은 flutter에서 오류가 많음
-> 그래서 json 파일로 변경
변경 방법 :
https://lottiefiles.com/ 여기서 MP4 파일을 JSON 파일로 변환
- Go to Dashboard 클릭
- 왼쪽 상단에 upload animations 클릭
- video & image to Lotties 클릭
- Browse 눌러서 GIF 파일 선택
- 오른쪽에 Lottie.JSON 클릭
- 다운받고 그 JSON 파일 이용하면 끝
(loading_character_male.json 파일은 배경에 흰색 제거하고,
loading_character_female.json 파일은 만들면 진행)

실시간으로 만들어지는 JSON 파일은 위의 폴더 경로 안에 있음
즉, insect_photos 폴더 안에 있음.

안드로이드 스튜디오에서 휴대폰 내부에 저장된 JSON 파일 보는 법
1) 왼쪽 상단에 가로 선 4개인거 클릭
2) view > Tool windows > Device explorer 클릭
3) 우측에 뜸
4) data/data/com.insect.cbnu/app_flutter/insect_photos/ 찾아가서 클릭
5) insect_photos 폴더 안에 JSON 파일로 저장되어 있음
